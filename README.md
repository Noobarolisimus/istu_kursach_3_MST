Алгоритмы Прима и Краскала для нахождения минимального остовного дерева.

# Requirements

- С++ 23 компилятор, предпочтительно clang
- VS 2022 / premake v5.0.0-beta7+, make

# Build and run

Я использую clang (v17) и далее он будет в примерах. Если вы используете visual studio, то стоит скачать clang. Это можно сделать через `Visual Studio Installer` или кнопки `Средства -> Получить средства и компоненты...`.

## Windows

Я подготовил два проекта в директории `project` для visual studio 2022: `Tests` и `Benchmarks` для юнит-тестов и проверки производительности соответсвенно.

Проект можно сгенерировать через premake, об этом ниже в секции про unix.

## Unix

Код разбит на два проекта, они генерируюся через premake в два решения соответственно. Это сделано из-за бага в premake на моей версии - под windows он пытается использовать линкер для linux.

Нужно установить [premake](https://https://premake.github.io), дальше для генераци проекта нужно прописать в терминал:
```premake5 --cc=clang --build_switch=benchmarks gmake```

- `--cc=` отвечает за компилятор
- `--build_switch=` отвечает за выбор либо бенчмарков(`benchmarks`), либо юнит-тестов(`tests`)
- На место `gmake` можно поставить другую build-систем, например, ninja, vs2012, vs2022 и [прочее](https://premake.github.io/docs/Using-Premake)

Далее стандартное использование make - прописать `make`
